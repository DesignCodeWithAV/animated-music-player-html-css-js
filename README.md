# Animated Music Player UI

A fully working animated music player built with **HTML, CSS, and JavaScript**.

This project is not just a static UI design. It includes **real audio playback**, **dynamic song switching**, **circular progress animation**, **rotating album art**, **moving clouds background**, and **custom playback controls**.

---

## Preview

This project features a modern and cinematic music player interface with:

- Animated cloud background
- Glassmorphism-inspired player card
- Circular SVG progress ring around album art
- Rotating album disc while music plays
- Real-time duration and current time update
- Custom play, pause, previous, and next controls
- Equalizer animation for a more interactive feel

---

## Features

- Real music playback
- Dynamic song title, artist name, and album cover
- Circular progress ring using SVG
- Live current time and total duration
- Previous and next track functionality
- Play and pause state animation
- Rotating album cover effect
- Animated equalizer bars
- Cinematic moving clouds background
- Clean and creative frontend UI

---

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
